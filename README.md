# IVE 팬 앱 모음 💜

IVE를 좋아하는 딸을 위해 만든 팬 앱 컬렉션입니다.

## 앱 목록

| 앱 | 파일 | 설명 |
|---|---|---|
| 💜 포토카드 리스트 | `index.html` | IVE 포토카드 보유/위시 관리 + 포카마켓·번개장터 시세 조회 |
| 👀 눈 맞추기 | `eyes.html` | IVE 멤버의 눈만 보고 누구인지 맞추는 퀴즈 |
| 💕 이상형 월드컵 | `worldcup.html` | 남녀 K-pop 아이돌/배우 이상형 월드컵 (사진 포함) |
| ⭐ 연예인 맞추기 | `celeb.html` | 초성+힌트로 한국 연예인 30명 이름 맞추기 |
| 💜 IVE 퀴즈 | `quiz.html` | IVE에 대한 50문제 퀴즈 |
| 📸 유진 사진 모음 | `yujin.html` | 안유진 사진 갤러리 (드래그&드롭, 태그, 즐겨찾기) |

## 기술 스택

- 순수 HTML/CSS/JavaScript (프레임워크 없음)
- 외부 API: pocali, phocamarket, pocamarket, bunjang
- 로컬스토리지 기반 데이터 저장
